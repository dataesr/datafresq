L'**espace actif** est l'espace de travail que vous avez sélectionné comme
destination pour l'ajout rapide de formations. Il est affiché en permanence en haut du
menu latéral, sous la mention **« Espace actif »**.

## Définir un espace actif

Deux méthodes :

- **Depuis le menu latéral :** cliquez sur le bouton de sélection dans la section « Espace actif ». Une fenêtre s'ouvre avec la liste de vos espaces. Cliquez sur celui que vous souhaitez activer.
- **Depuis le tableau de résultats :** cliquez sur **« Nouvel espace »**, puis sur **« Espace existant »** dans le panneau qui s'ouvre. Sélectionnez l'espace souhaité.

## Désélectionner l'espace actif

Cliquez sur le bouton de fermeture à côté de l'espace actif dans le menu latéral. Le bouton « Nouvel espace » réapparaîtra dans le tableau de résultats.

> [!NOTE]
> L'espace actif est mémorisé dans votre navigateur. Il reste sélectionné même si vous
> quittez la page ou fermez l'onglet, jusqu'à ce que vous le désélectionniez ou que
> l'espace soit supprimé.

## Comportement sur la page de recherche

Lorsqu'un espace actif est défini, le bouton
**« Ajouter à [nom de l'espace] »** remplace le bouton « Nouvel espace »
au-dessus du tableau de résultats. Vous pouvez alors
[ajouter des formations](/guide/espaces/ajouter) directement dans cet
espace sans passer par le formulaire de création.

Pour changer d'espace actif sans quitter la page de recherche, utilisez le sélecteur
d'espace accessible via le bouton de sélection dans le menu latéral.

L'onglet **« Paramètres »**, accessible uniquement au propriétaire de
l'espace, permet de configurer l'identité visuelle, la visibilité, les collaborateurs
et de supprimer l'espace.

## Général

Modifiez le nom, la description et la couleur de l'espace. Ces modifications sont
visibles immédiatement par tous les collaborateurs.

- **Nom** : identifie l'espace dans le menu latéral et les tableaux.
- **Description** : texte libre pour décrire le périmètre ou l'objectif de l'espace.
- **Couleur** : pastille colorée affichée dans le menu latéral pour distinguer visuellement vos espaces.

## Collaborateurs

Ajoutez de nouveaux collaborateurs en recherchant leur adresse email. Vous pouvez
modifier leur rôle (lecteur ou éditeur) ou les retirer de l'espace à tout moment.

Pour en savoir plus sur les rôles et permissions, consultez la page
[Partage et collaboration](/guide/espaces/partage).

## Visibilité

Basculez l'espace entre privé et public. Un espace public est consultable par tous les
utilisateurs connectés, mais seuls les collaborateurs invités peuvent le modifier.

> [!NOTE]
> Changer la visibilité d'un espace prend effet immédiatement. Si vous passez un espace
> de public à privé, les utilisateurs qui n'ont pas été explicitement invités perdent
> instantanément l'accès.

## Zone de danger

Vous pouvez supprimer définitivement un espace depuis cette section. Cette action est
irréversible et supprime toutes les données associées. Les collaborateurs perdent
immédiatement l'accès.

> [!WARNING]
> **Attention :** la suppression d'un espace est définitive. Assurez-vous
> d'avoir exporté les données nécessaires avant de procéder. Si d'autres collaborateurs
> travaillent sur cet espace, prévenez-les au préalable.

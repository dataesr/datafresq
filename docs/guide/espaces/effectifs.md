L'onglet « Effectifs étudiants » d'un espace de travail présente les données
d'inscription issues du système [SISE](/guide/donnees/sise), agrégées sur
l'ensemble des formations de l'espace. Un sélecteur en haut de page permet de
choisir une rentrée universitaire ou d'afficher la vue « Évolution » pour
comparer les données à travers les années.

## Indicateurs

Cinq cartes résument les données clés pour la rentrée sélectionnée :

| Carte | Description |
|-------|-------------|
| **Formations** | Nombre de formations ayant des données SISE pour cette rentrée, sur le total de formations dans l'espace |
| **Étudiants inscrits** | Somme des inscriptions principales de toutes les formations |
| **Femmes** | Nombre total de femmes inscrites |
| **Hommes** | Nombre total d'hommes inscrits |
| **Taux de féminisation** | Pourcentage de femmes parmi l'ensemble des inscrits |

> [!NOTE]
> Le taux de féminisation est calculé sur l'ensemble agrégé et non comme une moyenne
> des taux individuels.

Six graphiques présentent la ventilation des effectifs selon différentes dimensions.
Chaque graphique affiche le nombre d'étudiants inscrits (et non le nombre de formations) :

- **Par cycle** — répartition entre Licence, Master et Doctorat
- **Par type de diplôme** — Master, Licence professionnelle, BUT, Diplôme d'ingénieur…
- **Par académie** — effectifs par académie du siège de l'établissement
- **Carte par région** — carte choroplèthe colorée selon le nombre d'inscrits par région
- **Radar des grandes disciplines** — positionnement des effectifs par grande discipline (visible à partir de trois disciplines)
- **Par secteur disciplinaire** — granularité plus fine que le radar

Un tableau sous les graphiques liste les formations de l'espace avec leurs effectifs
pour la rentrée sélectionnée.

## Évolution

Lorsqu'au moins deux rentrées universitaires sont disponibles, la vue « Évolution »
présente un graphique de tendance avec trois séries :

- **Effectif total** — somme des inscriptions principales par année
- **Femmes** — évolution du nombre de femmes inscrites
- **Hommes** — évolution du nombre d'hommes inscrits

## Disponibilité des données

Cet onglet n'affiche des données que si au moins une formation de l'espace dispose de
données [SISE](/guide/donnees/sise). Si aucune formation n'est couverte,
un message indique l'absence de données.

> [!NOTE]
> Le nombre de formations avec des données SISE peut varier d'une rentrée à l'autre.
> Certaines formations récemment créées n'ont pas encore de données, tandis que des
> formations anciennes peuvent ne plus en recevoir.

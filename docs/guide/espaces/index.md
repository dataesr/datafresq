Un espace de travail vous permet de regrouper plusieurs formations pour analyser les
données  associées de manière agrégée, comparer des indicateurs et partager vos analyses avec d'autres utilisateurs de dataFresq. 
Cette fonctionnalité est réservée aux utilisateurs ayant un profil "Utilisateur +"

## Fonctionnement général

Un espace de travail contient une sélection de formations que vous constituez manuellement
ou à partir des résultats d'une recherche. Une fois peuplé, l'espace propose des analyses réalisées à partir des données associées à chacune des formations et agrégées entre elles, ainsi que des outils permettant de gérer le contenu et la configuration de l'espace de travail. Le tout est disponible dans plusieurs onglets thématiques permettant d'accéder facilement à l'information souhaitée

### Onglets d'analyse

Ces onglets présentent les données agrégées disponibles pour toutes les formations de l'espace :

| Onglet | Contenu |
|--------|---------|
| [**Offre de formation**](/guide/espaces/offre-de-formation) | Répartition par cycle, type de diplôme, secteur disciplinaire, répartition géographique, métiers ROME. |
| [**Effectifs étudiants**](/guide/espaces/effectifs) | Données issues du croisement Fresq x SISE et agrégées : effectifs totaux, répartition par genre, ventilation par cycle, diplôme, académie, région, discipline et évolution sur plusieurs années. |
| [**Insertion professionnelle**](/guide/espaces/insertion) | Données issues du croisement Fresq x InserSup et agrégées : taux d'emploi, distribution par formation, emploi stable, comparaison par genre et évolution par cohorte. |

### Onglets de gestion

Ces onglets permettent de gérer le contenu et la configuration de l'espace :

| Onglet | Contenu | Accès |
|--------|---------|-------|
| **Formations** | Liste des formations de l'espace, avec possibilité de les retirer individuellement. | Tous |
| **Historique** | Journal des actions effectuées par tous les membres de l'espace : créations, ajouts, suppressions, modifications de paramètres. | Propriétaire et éditeurs |
| [**Paramètres**](/guide/espaces/parametres) | Configuration de l'espace : nom, description, couleur, visibilité, collaborateurs, zone de danger. | Propriétaire uniquement |

Les tableaux de bord disponibles au sein d'un espace sont calculés en agrégeant les données disponibles dans chacune des
formations qu'il contient. Les effectifs sont sommés, les taux d'emploi sont des moyennes
pondérées par le nombre de sortants, et les répartitions sont agrégées par catégorie. Les
détails méthodologiques sont décrits dans la section
[Indicateurs et méthodologie](/guide/indicateurs).

## Organisation dans le menu latéral

Le menu latéral organise vos espaces de travail en trois sections :

- **Espace actif** : l'espace actuellement sélectionné pour l'ajout rapide de formations (voir [Espace actif](/guide/espaces/espace-actif)).
- **Mes espaces** : les espaces dont vous êtes propriétaire (hors espace actif). Vous pouvez plier/déplier cette section.
- **Partagés avec moi** : les espaces dans lesquels vous avez été invité en tant que lecteur ou éditeur. Cette section n'apparaît que si vous avez des espaces partagés.

Cliquez sur le nom d'un espace pour y accéder. Le nombre de
formations est indiqué sous le nom de l'espace.

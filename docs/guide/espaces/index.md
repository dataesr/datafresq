Un espace de travail vous permet de regrouper plusieurs formations pour analyser leurs
données de manière agrégée, comparer des indicateurs et partager vos analyses avec des
collaborateurs. C'est l'outil central de DataFresq pour travailler sur un périmètre de
formations personnalisé.

## Fonctionnement général

Un espace de travail contient une sélection de formations que vous constituez manuellement
ou à partir d'une recherche. Une fois peuplé, l'espace propose des onglets pour analyser
les données agrégées et gérer le périmètre.

### Onglets d'analyse

Ces onglets présentent les données agrégées de toutes les formations de l'espace :

| Onglet | Contenu |
|--------|---------|
| [**Offre de formation**](/guide/espaces/offre-de-formation) | Répartition par cycle, type de diplôme, secteur disciplinaire, répartition géographique, métiers ROME. |
| [**Effectifs étudiants**](/guide/espaces/effectifs) | Données SISE agrégées : effectifs totaux, répartition par genre, ventilation par cycle, diplôme, académie, région, discipline et évolution sur plusieurs années. |
| [**Insertion professionnelle**](/guide/espaces/insertion) | Données InserSup agrégées : taux d'emploi, distribution par formation, emploi stable, comparaison par genre et évolution par cohorte. |

### Onglets de gestion

Ces onglets permettent de gérer le contenu et la configuration de l'espace :

| Onglet | Contenu | Accès |
|--------|---------|-------|
| **Formations** | Liste des formations contenues dans l'espace, avec possibilité de les retirer individuellement. | Tous |
| **Historique** | Journal des actions effectuées sur l'espace : créations, ajouts, suppressions, modifications de paramètres. | Propriétaire et éditeurs |
| [**Paramètres**](/guide/espaces/parametres) | Configuration de l'espace : nom, description, couleur, visibilité, collaborateurs, zone de danger. | Propriétaire uniquement |

Les statistiques d'un espace sont calculées en agrégeant les données de toutes les
formations qu'il contient. Les effectifs sont sommés, les taux d'emploi sont des moyennes
pondérées par le nombre de sortants, et les répartitions sont agrégées par catégorie. Les
détails méthodologiques sont décrits dans la section
[Indicateurs et méthodologie](/guide/indicateurs).

## Organisation dans le menu latéral

Le menu latéral organise vos espaces en trois sections :

- **Espace actif** : l'espace actuellement sélectionné pour l'ajout rapide de formations (voir [Espace actif](/guide/espaces/espace-actif)).
- **Mes espaces** : les espaces dont vous êtes propriétaire (hors espace actif). Vous pouvez plier/déplier cette section.
- **Partagés avec moi** : les espaces auxquels vous avez été invité en tant que lecteur ou éditeur. Cette section n'apparaît que si vous avez des espaces partagés.

Cliquez sur le nom d'un espace pour accéder à sa page détaillée. Le nombre de
formations est indiqué sous chaque espace.

L'onglet « Offre de formation » donne une vue d'ensemble de la composition de votre
espace de travail. Il ne repose pas sur des données statistiques externes mais sur les
caractéristiques des formations elles-mêmes, issues du référentiel
[Fresq](/guide/donnees/fresq).

## Indicateurs

En haut de l'onglet, quatre cartes résument le périmètre de l'espace :

| Carte | Description |
|-------|-------------|
| **Formations** | Nombre total de formations dans l'espace |
| **Cycles LMD** | Nombre de cycles distincts représentés (Licence, Master, Doctorat) |
| **Régions** | Nombre de régions couvertes par les formations |
| **Types de diplômes** | Nombre de types de diplômes distincts (Master, BUT, Diplôme d'ingénieur…) |

Sous les cartes, des graphiques présentent la distribution des formations selon
différentes dimensions. Chaque graphique n'apparaît que si la dimension concernée
contient des données :

- **Cycle** — nombre de formations par cycle LMD (Licence, Master, Doctorat)
- **Type de diplôme** — nombre de formations par type de diplôme (Master, Licence professionnelle, BUT, Diplôme d'ingénieur…), plus fin que le cycle
- **Académie** — nombre de formations par académie
- **Région** — carte choroplèthe de la France métropolitaine colorée selon le nombre de formations par région
- **Métiers ROME** — diagramme radar des grands domaines professionnels ROME associés aux formations de l'espace (visible uniquement si des formations disposent de codes [ROME](/guide/donnees/rome) via le RNCP)
- **Secteur disciplinaire** — nombre de formations par secteur disciplinaire, issu de la classification [SISE](/guide/donnees/sise)

## Disponibilité des données

Cet onglet ne repose pas sur des données statistiques (effectifs, insertion) mais sur les
métadonnées des formations. Il est donc toujours disponible dès qu'un espace contient au
moins une formation, sans condition de seuil ni de couverture par une source externe.

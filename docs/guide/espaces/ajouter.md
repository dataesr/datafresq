Lorsqu'un [espace actif](/guide/espaces/espace-actif) est défini, le
bouton **« Ajouter à [nom de l'espace] »** remplace le bouton
« Nouvel espace » au-dessus du tableau de résultats. En cliquant dessus, un panneau
s'ouvre avec une prévisualisation de l'ajout.

## Sélection manuelle

Cochez les formations souhaitées dans le tableau à l'aide des cases à cocher, puis
cliquez sur le bouton d'ajout. Seules les formations sélectionnées seront ajoutées.

## Ajout par recherche

Sans sélection manuelle, vous pouvez ajouter l'ensemble des résultats correspondant à
votre recherche et vos filtres. Cochez la case
**« Je souhaite ajouter tous les résultats de la recherche »**, puis
cliquez sur le bouton d'ajout.

## Prévisualisation

Avant de confirmer, le panneau vous indique :

- Le nombre total de formations concernées
- Le nombre de formations déjà présentes dans l'espace (elles seront ignorées)
- Le nombre de formations qui seront effectivement ajoutées

Un avertissement s'affiche pour les ajouts volumineux (plus de 1 000 formations).

> [!WARNING]
> **Limite :** un espace peut contenir jusqu'à
> **5 000 formations**. Au-delà de cette limite, le bouton d'ajout est
> désactivé. Réduisez le périmètre de votre recherche ou créez un second espace.

## Retirer des formations

Pour retirer des formations d'un espace, rendez-vous sur la page de l'espace, onglet
**« Formations »**. Chaque ligne du tableau propose un bouton de
suppression pour retirer la formation individuellement.

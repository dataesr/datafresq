Trois méthodes sont disponibles pour créer un espace de travail selon votre contexte :
depuis le menu latéral, depuis la page Espaces ou directement depuis les résultats de
recherche.

## Depuis le menu latéral

Dans la section **« Mes espaces »** du menu latéral, cliquez sur le
bouton **+**. Vous serez redirigé vers le formulaire de création.

## Depuis la page Espaces

Rendez-vous sur la page [Espaces](/espaces) et cliquez sur la carte
**« Créer un espace »**.

## Depuis les résultats de recherche

Lorsqu'aucun espace actif n'est défini, le bouton
**« Nouvel espace »** apparaît au-dessus du tableau de résultats de la
page [Formations](/formations).

1. Cliquez sur **« Nouvel espace »** pour ouvrir le panneau de prévisualisation.
2. Vérifiez le nombre de formations concernées. Si vous n'avez pas sélectionné de formations individuellement, cochez la case **« Je souhaite ajouter tous les résultats de la recherche »**.
3. Cliquez sur **« Créer un espace »** pour accéder au formulaire de création, pré-rempli avec votre recherche.
4. Renseignez le nom, la description, la couleur et les options de visibilité, puis validez.

> [!WARNING]
> **Astuce :** lorsque vous créez un espace depuis la recherche, deux
> boutons vous sont proposés : **« Créer et voir l'espace »** pour accéder
> directement à votre nouvel espace, ou
> **« Créer et revenir à la recherche »** pour continuer à explorer les
> formations.

## Formulaire de création

Le formulaire de création vous propose les options suivantes :

| Champ | Description |
|-------|-------------|
| **Nom** | Obligatoire. Donnez un nom descriptif à votre espace (ex : « Masters IA Île-de-France »). |
| **Description** | Optionnel. Un texte libre pour décrire le périmètre ou l'objectif de l'espace. |
| **Couleur** | Choisissez une couleur pour identifier visuellement l'espace dans le menu latéral et les tableaux. |
| **Visibilité** | Privé (seuls vous et les collaborateurs invités y accèdent) ou Public (visible par tous les utilisateurs). |
| **Collaborateurs** | Invitez d'autres utilisateurs dès la création. Vous pourrez en ajouter plus tard dans les [paramètres](/guide/espaces/parametres). |

L'onglet « Effectifs étudiants » présente les données d'inscription issues du
système [SISE](/guide/donnees/sise) : effectif total, répartition par
genre et évolution sur plusieurs années universitaires.

## Indicateurs

En haut de l'onglet, des cartes de synthèse résument les données clés pour l'année sélectionnée :

| Carte | Description |
|-------|-------------|
| **Etudiants inscrits** | Nombre total d'étudiants inscrits |
| **Hommes** | Nombre d'étudiants inscrits |
| **Femmes** | Nombre d'étudiantes inscrits |
| **Taux de féminisation** | Pourcentage de femmes parmi les étudiants inscrits |

> [!NOTE]
> Les effectifs affichés sont les effectifs en inscription principale y 
> compris les doubles inscriptions CPGE (étudiants de classes préparatoires inscrits
> en parallèle à l'université). Pour plus de détails,
> consultez la page [Indicateurs — Effectifs](/guide/indicateurs/effectifs).

Les données SISE sont ventilées selon les dimensions
suivantes, chacune déclinée par genre dans des graphiques en barres :

- **Commune** — Distribution des étudiants selon leur commune d'inscription
- **Année d'étude** — Distribution des étudiants selon leur année dans le cursus

## Évolution

Lorsqu'au moins deux années universitaires sont disponibles, un graphique présente
l'évolution des effectifs avec les tendances suivantes :

- **Effectif total** — inscriptions principales par année
- **Femmes** — inscriptions principales par année
- **Hommes** — inscriptions principales par année

Ces graphiques permettent d'observer les tendances d'attractivité de la formation sur
plusieurs années et de repérer d'éventuelles variations significatives.


## Disponibilité des données

Cet onglet n'est disponible que si des données SISE existent pour cette formation.
Certaines formations récentes ou non couvertes par SISE n'afficheront pas cet onglet.

L'appariement entre les formations Fresq et les données SISE repose sur les codes SISE
associés à la formation et ses parcours. Si ces codes sont absents ou incorrects, les
données ne peuvent pas être reliées.

Dans le tableau de recherche, le filtre
[« Données SISE disponibles »](/guide/recherche/filtres) permet
d'isoler les formations couvertes.

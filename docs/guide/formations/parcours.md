L'onglet « Parcours » détaille l'organisation pédagogique de la formation : parcours
(spécialisations, options), étapes et lieux de formation, année par année.

## Sélection de l'année

Les parcours et étapes peuvent évoluer d'une année à l'autre. Un sélecteur d'année vous
permet de consulter l'organisation de la formation pour chaque rentrée disponible. Par
défaut, l'année la plus récente est affichée.

## Vue d'ensemble

En haut de l'onglet, des cartes de synthèse résument les données clés de l'ensemble des
parcours pour l'année sélectionnée :

| Carte | Description |
|-------|-------------|
| **Parcours** | Nombre total de parcours (ouverts et fermés) |
| **Fermés** | Nombre de parcours fermés pour l'année sélectionnée |
| **Capacité d'accueil** | Somme des capacités déclarées sur tous les parcours |
| **Alternance** | Nombre de parcours proposant l'alternance |
| **Distanciel** | Nombre de parcours proposant un enseignement à distance |
| **Sites** | Nombre de sites de formation distincts |

## Parcours

Chaque parcours est présenté sous forme d'une carte dépliable contenant :

- **Intitulé et sigle** du parcours
- **Statut** : ouvert ou fermé pour l'année sélectionnée
- **Identifiants** : codes SISE et RNCP associés au parcours
- **Étapes** : les étapes pédagogiques composant le parcours (voir ci-dessous)

## Étapes

Chaque étape représente une année ou un niveau dans le parcours. Une carte d'étape
peut contenir les informations suivantes selon leur disponibilité :

- **Informations pédagogiques** : disciplines, langues d'enseignement, attendus et
  prérequis, diplômes recommandés, spécialités
- **Recrutement** : capacité d'accueil, modalités de recrutement
- **Modalités d'enseignement** : formation initiale, alternance, distanciel, formation
  continue
- **Lieux de formation** : adresses et coordonnées géographiques, affichées sur une
  mini-carte lorsque les données de géolocalisation sont disponibles

## Disponibilité des données

Cet onglet n'est visible que si la formation contient au moins un parcours. Le nombre
de parcours est indiqué entre parenthèses dans le libellé de l'onglet.

Les données de parcours proviennent de la base [Fresq](/guide/donnees/fresq)
et ne sont actuellement disponibles que pour les diplômes de type **Master** et **BUT**.

> [!NOTE]
> L'organisation en parcours et étapes reflète la structure déclarée par les
> établissements dans le référentiel Fresq.
